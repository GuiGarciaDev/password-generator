# Password-generator

Website for generating random password with different types of char, you can include uppercases, lowercases, numbers and symbols.
   
## Tools and Libraries
    
* Tools:
    * React.js 
    * Typescript
    * Sass
    * Firebase Hosting

* Libraries: 
    * react-icons
